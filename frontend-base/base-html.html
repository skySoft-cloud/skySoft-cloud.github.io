# 为什么推出HTML5?

  1. WEB浏览器之间的兼容性很低
  2. 文档结构不够明确
  3. WEB应用程序的功能受到限制
  4. HTML5的设计目的是为了在移动设备上支持多媒体

# HTML5 新增特性

# 结构标签

  1. 结构标签：主要用于页面骨架，布局相关，例如一个页面包含顶部、导航、页脚等;
  • header:一般是网页的头部信息/标题。
  • nav:导航条部分内容。
  • section:独立内容区。
  • article:独立内容区中的小单元。
  • aside:标签内容之外，与标签内容相关的辅助信息。
  • footer:底部信息。 代码示例：
<header>顶部信息</header>
<nav>导航信息</nav>
<div>
  <aside>侧边信息</aside>
  <section>
    <article>内容区</article>
  </section>
</div>
<footer>页面底部</footer>

# 表单标签

  1. 表单标签：H5新增的多是表单中input 的type 属性，具体属性值如下：
  • number:只能输入数值，能通过 min/max设置最大最小值。
  • range:必须输入一个范围内的数值。
  • email:自动校验邮箱地址。
  • url:自动校验 HTTP地址。
  • 日期选择器:可选属性值有 date：选取年月日。month：选取年月。week：选择周和年。time：选取时间。datetime：选取时间、年月日。
  • color:颜色选择器。

# 媒体标签

  1. 媒体标签：主要为音频、视频等标签；
  • video:视频标签。
  • audio:音频标签。

# 本地存储

  1. 本地存储：这里指的web存储；
  • localStorage:本地存储。
  • sessionStorage:会话存储。

# 其它新特性

  • 拖拽释放 API:(Drag and drop) API。
  • 离线应用:manifest。
  • 桌面通知:Notifications。
  • 地理位置:Geolocation。
  • 多任务 webWorker:JS可以有多个线程，子线程受主线程控制，并且不能操作DOM，主线程可以操作DOM，所以JS依然是单线程。
  • 全双工通信协议:websocket。
  • 历史管理:history。
  • 绘画:canvas。
